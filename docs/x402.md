# 📄 `aptos-x402-integration.md`

## Overview

`aptos-x402` is a **TypeScript SDK that enables pay-per-request APIs using HTTP 402 (“Payment Required”) + blockchain micropayments on Aptos** ([npmjs.com][1])

It allows you to:

* Charge per API call (no subscriptions, no API keys)
* Let **AI agents or users pay automatically**
* Embed **payment logic directly into HTTP requests** ([boostylabs.com][2])

## ⚙️ How aptos-x402 Works (Simplified Flow)

1. Client requests API
2. Server responds:

```
402 Payment Required
```

3. Client signs blockchain payment
4. Retries request with:

```
X-PAYMENT: <signed payload>
```

5. Server verifies → returns response

👉 No Stripe, no auth, no billing system
👉 Pure protocol-level payments ([Medium][3])

### 2. Add Middleware to Your API

```ts

app.use(
  paymentMiddleware(
    process.env.PAYMENT_RECIPIENT!,
    {
      "/api/voice/process": {
        price: "1000000", // 0.01 APT
        network: "testnet",
        config: {
          description: "Voice processing & cloning request",

        price: "500000", // 0.005 APT

  )
);
```

## 🤖 VoiceVault-Specific Design (Important)

### Add “Payment Awareness” to Agents

Extend your frontend/agent request interceptor (e.g., in `api.ts`):

```ts
if (response.status === 402) {
  // 1. Extract payment requirement from headers
  const req = await response.json();
  
  // 2. Prompt Aptos wallet (AptosWalletAdapter) to sign the transaction
  const payment = await wallet.signAndSubmitTransaction(req.transactionData);
  
  // 3. Retry the request with the payment signature
  retryWithPayment(payment.hash);
}
```
